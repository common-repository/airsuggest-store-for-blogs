By using this plugin you can add Airsuggest store on your site.

== Description ==
1. Any Store - Ecommerce - Add a complete 8,000 Products Discounted Ecommerce store on your Blog which opens only when a user clicks on ‘explore store’ and is universal store. Get 10 - 30 % Commision on each sale 
2. Give over 1500 featured deals to your user.Once again, it open only when a user clicks on ‘explore store’ button
3. Sell your own products and subscription by making your own store and cross sell your products across 10,000 more blogs.Once again, It opens only when a user clicks on ‘explore store’ and is universal store.

Note that you are signed up as vendor in the site http://airsuggest.com/v_register.php and then you get an api key. using this api key you get you store iframe and external scripts with url http://airsuggest.com/page-shop.php in your wordpress site.

Ask any query e-mail: admin[at]airsuggest.com

== Installation ==

1. Upload the plugin files to the '/wp-content/plugins/airsuggest-blog' directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. The js will work automatically
